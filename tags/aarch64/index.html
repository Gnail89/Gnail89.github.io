VM installation for aarch64 on x86_64 host

VM installation for aarch64 on x86_64 host 案例一，centos 7运行aarch64虚拟机，完成ntp软件编译 1. 基础环境 基础操作系统: CentOS 7 x86_64 虚拟机操作系统: CentOS 7 aarch64 # 安装CentOS 7基础操作系统。并下载qemu源码包。 wget https://download.qemu.org/qemu-5.2.0.tar.xz #qemu源码 wget http://releases.linaro.org/components/kernel/uefi-linaro/16.02/release/qemu64/QEMU_EFI.fd #aarch64 UEFI引导程序 # 安装libvirt管理软件 yum install libvirt virt-manager systemctl start libvirtd 2. 编译qemu yum install Ninja-build gcc gcc-c++ bzip2 tar xvJf qemu-5.2.0.tar.xz cd qemu-5.2.0 ./configure --target-list=aarch64-softmmu make && make install 3. 安装aarch64虚拟机 # 准备镜像：CentOS-7-aarch64-Minimal-2009.iso # 创建卷 qemu-img create -f qcow2 centos7_aarch64.qcow2 10G # 启动虚拟机 qemu-system-aarch64 -m 2048 -cpu cortex-a72 -smp 2 -M virt \ -bios QEMU_EFI....
