OpenStack example#

# 将image镜像设置为vga显示驱动，以支持更大分辨率
openstack image set --property hw_video_model=vga image_id 

# 将volume卷设置为vga显示驱动，以支持更大分辨率（已有虚拟机需要重建）
openstack volume set --image-property hw_video_model=vga volume_id
